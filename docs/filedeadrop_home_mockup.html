FAQ
Can I recover a file after the link is accessed?
No. Deletion is immediate and permanent. This is by design.
Can you read my files?
No. Files are encrypted in your browser using a key that only exists in the share link. We never have access to it.
What if the link expires before my recipient opens it?
The file and link are permanently deleted after 24 hours regardless of whether it was accessed.
What if you receive a legal request?
We can produce an encrypted, unreadable file and basic transfer metadata. We have no access to file contents, filenames, or identities.
What file types are supported?
Any file type up to 100MB. We do not inspect, process, or scan file contents.
